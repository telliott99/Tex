===============
1. Substitution
===============

# 1
int sin(ln x)/x dx

u = ln x
du = 1/x dx

= int sin(u) du
= - cos u + C
= - cos (ln x) + C

# 2
int sin x cos x dx

u = sin x
du = cos x dx

= int u du
= 1/2 u^2 + C
= 1/2 sin^2 x + C

note:
sin^2 x = 1/2 (1 - cos 2x)
so above can be written:
= 1/4 (1 - cos 2x) + C

# 3
int x/sqrt(1 - x^2) dx

u = 1 - x^2
- 1/2 du = x dx

= int 1/sqrt(u) du
= 2 sqrt(u) + C
= 2 sqrt(1 - x^2) + C

# 4
int 1/(x ln x) dx

u = ln x
du = 1/x dx

= int 1/u du
= ln u + C
= ln (ln x) + C

# 5
int x sin(x^2) dx

u = x^2
1/2 du = x dx
 
= int sin u du
= - cos u + C
= - cos (x^2) + C

harder:
# 6

int sqrt(4 - sqrt(x)) dx

u = 4 - sqrt(x)
du = - 1/(2 sqrt(x)) dx

= (-2) int sqrt(u) sqrt(x) du
= (-2) int sqrt(u) (4 - u) du
= (-2) [ (4) 2/3 u^3/2 - 2/5 u^5/2 ] + C
= -16/3 (4 - sqrt(x))^3/2 + 4/5 (4 - sqrt(x))^5/2 + C

cos(s+t) = cos s cos t - sin s sin t

====================
5. Partial Fractions
====================

Can the denominator be factored?
Example:

  2x         1       1
-------  = ----- + -----
x^2 - 1    x - 1   x + 1

   2         1       1
-------  = ----- + -----
1 - x^2    1 - x   1 + x

generally, write

      c             A         B 
-------------- = ------- + -------
(x - 1)(x + 1)   (x - 1)   (x + 1)

A (x + 1) + B (x - 1) = c

must be true for *all* x

A = -B
2A = c

Can be substantially more complicated.
See:

https://www.math.ucdavis.edu/~kouba/CalcTwoDIRECTORY/
partialfracdirectory/PartialFrac.html
